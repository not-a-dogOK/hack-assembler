# hack-assembler
an assembler for the hack machine language
knows to handle A type instructions (op code 0), C type instructions (op code 1) and L type instructions (lable for easier usage)

# API
run a line in the follwing order: 
1) gets line.
2) fr pass on line. 
3) sc pass on line.
